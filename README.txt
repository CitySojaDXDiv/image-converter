========================================
画像フォーマット一括変換ツール
完全オフライン対応ポータブル版
========================================

【使い方】
1. index.htmlをダブルクリックして開く
2. ブラウザで動作します（Chrome/Edge/Safari推奨）
3. インターネット接続は不要です

【動作環境】
- Windows 10/11
- macOS 10.15以降
- 最新のブラウザ（Chrome/Edge/Safari）

【機能】
- JPEG/PNG/WebP/AVIF/BMP相互変換
- 一括処理（最大50枚）
- 品質調整
- リサイズ（オプション）
- 変換後サイズのプレビュー
- ZIP一括ダウンロード

【対応フォーマット】
- 読み込み: JPEG, PNG, WebP, AVIF, BMP
- 出力: JPEG, PNG, WebP, AVIF, BMP

【ファイル構成】
image-converter-offline/
├── index.html          # メインファイル（これを開く）
├── style.css           # スタイル
├── main.js             # プログラム
├── libs/               # ライブラリ（オフライン動作用）
│   ├── jszip.min.js
│   └── FileSaver.min.js
├── manual.html         # 取扱説明書
└── README.txt          # このファイル

【注意事項】
- ファイル構成を変更しないでください
- libs/フォルダは削除しないでください
- 大量データ（50枚以上）は処理に時間がかかります
